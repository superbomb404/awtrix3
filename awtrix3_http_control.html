状态与屏幕

读取设备统计、效果列表、过渡效果、循环应用和当前矩阵画面。LiveView 链接会直接打开设备内置页面。

设备统计 GET /api/stats 电量、RAM、当前 App 等设备状态。
效果与过渡 GET /api/effects /api/transitions 用于自定义 App 背景效果和全局切换动画。
循环应用 GET /api/loop 读取当前轮播队列中的 App。
屏幕快照 GET /api/screen 读取 24-bit 色值数组，并渲染到上方矩阵预览。
LiveView /screen /fullscreen 打开 AWTRIX 内置实时画面页面。

电源与声音

控制矩阵开关、让设备进入深度睡眠、播放 MELODIES 文件夹内的声音，或直接发送 RTTTL 字符串。

矩阵电源 POST /api/power 发送 {"power": true/false}。
深度睡眠 POST /api/sleep 设备只能按时间或中键唤醒，API 无法远程唤醒。
播放声音文件 POST /api/sound RTTTL 文件名无需扩展名；DFPlayer 可填 4 位 MP3 编号。
发送 RTTTL POST /api/rtttl 直接发送 RTTTL 字符串正文。

灯光与指示

设置整屏氛围灯，或调整右侧三枚彩色指示灯。指示灯可附加 blink / fade 毫秒间隔。

氛围灯 POST /api/moodlight 可使用 RGB 颜色或 Kelvin 色温；关闭时发送空 body。
指示灯 1 POST /api/indicator1 右上角。
指示灯 2 POST /api/indicator2 右侧中部。
指示灯 3 POST /api/indicator3 右下角。

自定义 App 与通知

按官方 JSON 属性构建 payload。勾选“包含”后才会发送该字段；编辑字段会自动勾选。高级 JSON 可用于彩色文本片段、多页面数组或更复杂的 draw 数据。

发送目标 /api/custom?name= /api/notify Custom App 需要 name；Notify 不需要。
JSON 预览 Payload 发送前可检查最终 body。

维护操作

            
包含更新、重启、格式化与清空 Settings API 设置。危险动作会二次确认。

          

        

        

          

            

              

                触发更新
                POST /api/doupdate
                文档标注可发送 JSON；为空时发送空 body。
              

            

            

              
              
            

            

              
            

          

          

            

              

                重启设备
                POST /api/reboot
                发送空 payload。
              

            

            

              
            

          

          

            

              

                格式化 Flash 与 EEPROM
                POST /api/erase
                不会修改 Wi-Fi 设置，但相当于设备数据层面的恢复出厂。

                POST /api/resetSettings
